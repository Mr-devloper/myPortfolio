# Mizanur Rahman — Portfolio (Redesign 2024)

Modern redesign of the original portfolio. Dark editorial aesthetic with smooth animations.

## Stack
- Pure HTML, CSS, JavaScript (no frameworks)
- Google Fonts: Syne, DM Sans, Space Mono
- GitHub Pages ready

## Features
- Splash screen with typewriter animation
- Sticky navbar with scroll detection + active state
- Mobile hamburger menu
- Hero section with animated reveal
- About section with image + stats
- Animated skill progress bars (scroll-triggered)
- Contact cards
- Floating social links
- Fully responsive

## Deploy to GitHub Pages
1. Push all files to a GitHub repository
2. Go to **Settings → Pages**
3. Set source to `main` branch, `/ (root)` folder
4. Your site will be live at `https://yourusername.github.io/repo-name`
